# Mobile Data Collection App – User Guide

## Introduction
This application helps field consultants collect and manage farmer data digitally. It simplifies field data entry and reduces reliance on manual documentation.

---

## Getting Started

1. Open the application on your mobile device  
2. Log in using your assigned credentials  
3. You will be redirected to the dashboard  

---

## Recording New Data

1. Navigate to “New Entry”  
2. Enter required details:
   - Farmer ID  
   - Name  
   - Crop type  
   - Land size  
   - Location  
3. Review the entered information  
4. Tap “Submit”  

A confirmation message will appear once the data is saved.

---

## Viewing Existing Records

1. Go to the “Records” section  
2. Apply filters if required  
3. Select a record to view details  

---

## Editing Records

1. Open the required record  
2. Tap “Edit”  
3. Make necessary changes  
4. Save updates  

---

## Troubleshooting

**Issue: Unable to log in**  
- Verify credentials  
- Check internet connection  

**Issue: Data not saving**  
- Ensure all required fields are filled  
- Retry after checking connectivity  

**Issue: App loading slowly**  
- Restart the application  
- Check network strength  

---

## Best Practices

- Enter data during or immediately after field visits  
- Double-check entries before submission  
- Sync data regularly in low-network areas  

---

## FAQs

**Can I edit a record after submission?**  
Yes, records can be updated from the “Records” section.

**Is internet required?**  
Yes, for submitting and syncing data.
